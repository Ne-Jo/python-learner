### <kbd>**print()**</kbd>函数

#### 作用：告诉计算机，把括号里的内容显示在屏幕上

* 不带引号

```python
# 让计算机读懂括号里的内容，打印最终结果
>>>print(1+1)
2
```

* 带引号

```python
# 让计算机原样输出引号中的内容
>>>print('千寻')
千寻
>>>print("千寻")
千寻
```

* 带双引号

~~~python
# 与单引号作用一致，双引号可以输出内容中的单引号
>>>print("Let's go")
Let's go
~~~

* 带三引号

~~~python
# 实现换行
>>>print('''我愿意留在汤婆婆的澡堂里工作两年，
~~~

### 变量和赋值

#### 变量：带标签的空盒，赋值：把数据放入带标签的空盒。

~~~python
>>>name='千寻'
>>>name='小千'
>>>print(name)
小千	# 最终输出的结果是最后赋值给变量的值
~~~

* 只能是一个词；
* 只能包含字母、数字和下划线；
* 不能以数字开头；
* 尽量描述包含的数据内容（见名知义）；
* 不要使用函数名或关键字。


### 数据类型

#### 字符串<kbd>**str**</kbd>

> <kbd>**String**</kbd>简写<kbd>**str**</kbd>
>
> 由单引号、双引号、三引号括起来的文本
>
> ~~~python
> '千寻'
> "千寻"
> '''千寻'''
> >>> Chinese = '镜像世界'
> >>> English = "mirror world"
> >>> number = "666"
> >>> symbol = '''科A!@'''
> >>> mixture = '镜像世界mirror world666科A!@'
> >>> 
> >>> print(Chinese)
> 镜像世界
> >>> print(English)
> mirror world
> >>> print(number)
> 666
> >>> print(symbol)
> 科A!@
> >>> print(mixture)
> 镜像世界mirror world666科A!@
> ~~~

#### 整数<kbd>**int**</kbd>

> <kbd>**integer**</kbd>简写<kbd>**int**</kbd>
>
> 正整数、负整数和零的统称，没有小数点的数字
>
> ~~~python
> >>> # 打印整数
> ... print(333)
> 333
> ~~~

#### 浮点数<kbd>**float**</kbd>

> <kbd>**float**</kbd>无简写
>
> 比整数多了小数点<kbd>**.**</kbd>的数，运算结果会有误差
>
> ~~~python
> >>> print(0.55+0.3)
> 0.8500000000000001
> ~~~
>
> 进制转换出现的误差

### 数据的应用

#### 四则运算

##### Python 算数运算符

| 运算符 |          表示           |  例子   |
| :----: | :---------------------: | :-----: |
|   +    |           加            |  2+1=3  |
|   -    |           减            |  2-1=1  |

> 将其他数据数据类型转成整数

### <kbd>**if**</kbd>嵌套

#### 应用场景：

在基础条件满足的情况下，再在基础条件底下增加额外的条件判断。

#### <kbd>**if**</kbd>嵌套的执行顺序

注意：<kbd>**elif**</kbd>也是可以放进嵌套里的

~~~
考试成绩评价规则：

1. 如果成绩大于等于60分，就是及格，在此前提下：
(1)如果成绩大于等于80分，属于优秀范围；
(2)否则（即大于等于60分而小于80分），属于一般范围。

2. 如果成绩小于60分，就是不及格，在此前提下：
(1)如果成绩小于30分，平时太不认真，属于学渣了；
(2)如果成绩（小于60分而大于等于30分），那么，至少还能抢救一下。
~~~

代码：

~~~python
score=29	# 赋值

if score>=60:	# 条件1:及格的情况
	print('你及格了')
	if score>=80:	# 条件1-1:优秀情况
		print('你很优秀')
	else:			# 条件1-2
		print('一般')
else:			# 条件2:不及格情况
	print('不及格')
	if score<30:	# 条件2-1:学渣情况
		print('学渣')
	else:			# 条件2-2:可以抢救情况
		print('还可以抢救一下')
	print('查询完毕')
~~~
